## Practical Activities Completed

During the SimuLearn exercises, I worked with AWS S3 and performed object management tasks such as:

- Uploading files to an S3 bucket
- Renaming/modifying object names inside S3
- Understanding how object storage works
- Learning how S3 stores and organizes files using buckets and object keys
- Exploring basic cloud storage management concepts

This helped me understand how cloud storage systems work in real environments and how object-based storage differs from traditional local file systems.

## AWS Services Used

### Amazon S3
Used Amazon S3 to:
- Upload and manage objects
- Modify object names (object keys)
- Understand bucket structure
- Explore object storage concepts
- Learn how cloud-based storage is managed in AWS

I also learned that Amazon S3 does not perform true file renaming internally. Instead, objects are copied to a new key and the old object is removed.
