# Linux Fundamentals

## Overview
This learning covered the fundamentals of the Linux operating system, including its core concepts, user interfaces, documentation system, and Linux distributions. The goal was to build a foundational understanding of how Linux works and why it is widely used in servers, cloud computing, cybersecurity, and software development.

---

## What I Learned

### 1. What Linux Is
Linux is an open-source operating system based on Unix principles. It is widely used because of its stability, security, flexibility, and performance.

Linux is commonly used in:
- Cloud Computing
- Web Servers
- Cybersecurity
- Networking
- Software Development
- Embedded Systems

---

### 2. Main Features and Components of Linux

#### Key Features
- Open-source
- Multi-user support
- Multitasking
- Strong security model
- High stability and reliability
- Customizable environment

#### Core Components
- **Kernel**  
  The core of the operating system that manages hardware and system resources.

- **Shell**  
  The command-line interface that allows users to interact with the system.

- **File System**  
  Organizes and stores files and directories.

- **Processes**  
  Running programs managed by the operating system.

- **Utilities and Commands**  
  Built-in tools used for administration and daily tasks.
